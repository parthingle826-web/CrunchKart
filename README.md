# 🍌 CrunchKart - Banana Chips Website

CrunchKart is a modern web application built using Vite and React that showcases and sells delicious banana chips and wafers.

## 🚀 Features

- 🏠 Home page with attractive UI
- 📄 About page with brand story
- 🛒 Products page with pricing & discounts
- 🧾 Cart system (Add to Cart, Quantity control)
- 📦 Checkout with user details
- 🧾 Receipt generation with brand logo
- 📞 Contact page with form & business info

## 🛠️ Tech Stack

- ⚛️ React (Vite)
- 🎨 CSS (Custom styling)
- 📦 JavaScript (State management)
